# Avanti - a NODEJS library for simple local HTTPS hosting

A minimal JavaScript library.

## Installation

```bash
npm install avanti
```

## Usage

```javascript
import { hello } from 'avanti';

console.log(hello('World'));
```

## API

### `hello(name)`

Returns a greeting message.

- `name` (string, optional): Name to greet. Defaults to 'World'.
